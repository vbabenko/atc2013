Automation Test Course 2013

Информация про курс

Курс автоматизации web-приложений предназначен для тех, кто хочет получить свой первый рабочий опыт в IT. В первую очередь мы ориентируемся на студентов 4-5 курсов, которые не только имеют желание связать свою карьеру с разработкой ПО, но и прикладывают для этого усилия. Занятия будут проходить на бесплатной основе в офисе компании Levi9 Ukraine.

Весь учебный период будет разбит на 2 этапа: первый и второй Ыыыы. На первом будут преподаваться основы тестирования ПО. На втором этапе начнем работать с автоматизацией web-приложений на Java + WebDriver.

Первое занятие состоится 07.10.2013 в 19:00. В последующие 6 недель занятия будут проходить 3 раза в неделю (Пн, Ср, Пт) в 19:00.

Информация для участников

Для всех желающих принять участие в Automation Test Course 2013 открыта регистрация до 29.09.2013. Отправляйте свои заявки с нашего сайта, чтобы попасть на собеседование.

Каждый претендент индивидуален и обладает своими человеческими качествами. Для нас наиболее важны:

  • Ответственность
  • Настойчивость
  • Упорство
  • Пунктуальность

Существенным преимуществом будут знания в следующих технических областях:

  • Java
  • HTML
  • CSS

Разумеется, что эти пункты уже являются крепким фундаментом для развития и обучать людей с таким багажем знаний куда проще. Но не взирая на это, предпочтительным остаются именно человеческие качества студентов.
